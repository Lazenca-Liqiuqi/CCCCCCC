本目录为独立的 Git 仓库，用于跟踪文章翻译的版本变更。

**仓库信息**：
- **分支**：main
- **用途**：独立管理翻译文章的版本历史
- **提交规范**：每次新增或修改翻译文章需单独提交

## 翻译规范

1. **术语一致**：遵循根目录 `.claude/terminology.md` 的术语表
2. **双语对照**：采用原文与译文对照的格式
3. **质量保证**：确保翻译准确、通顺、符合中文表达习惯
4. **格式统一**：保持 Markdown 格式的规范性和一致性
