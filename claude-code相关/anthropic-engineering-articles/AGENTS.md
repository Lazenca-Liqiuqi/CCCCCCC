# Codex

## 目录定位

- 本目录保存 Anthropic Engineering 相关文章的中文翻译与整理结果
- 这些文章的主题可以是 Claude、Claude Code、Agent SDK 或相关工程实践
- 这些正文内容属于项目资料，不等同于仓库自身的全局记忆组件

## 文件组织

- 文章按编号顺序组织，保持稳定编号
- 新增文章时沿用当前编号风格与双语排版方式
- 与本目录相关的全局规则仍遵循 `.claude/rules/translation-format.md`

## 维护约定

- 优先维护标题、发布时间、原文链接与双语对应关系
- 不在本文档记录逐次修改历史，变更由 `git` 与根目录 `CHANGELOG.md` 承担
- 如果本目录后续新增 Codex 主题文章，可按现有结构继续扩展
